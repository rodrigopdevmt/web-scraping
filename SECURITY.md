# Segurança

## Reportando vulnerabilidades

Valorizamos a segurança da nossa comunidade. Se você encontrar uma vulnerabilidade, **não abra uma issue pública**.

Envie um email para **developer@rodrigodevmt.com.br** com:

- Descrição do problema
- Passos para reproduzir
- Possível impacto

Responderemos em até 48 horas.

## Práticas de segurança

- Não comite senhas, tokens ou chaves
- Use variáveis de ambiente para configurações sensíveis
- Mantenha as dependências atualizadas
